# Context

## Taxemotor domain terms

- **Anläggningsavgift**: engångsavgift för anslutning till allmän VA-anläggning.
- **Brukningsavgift**: löpande avgift för nyttjande av VA-tjänster.
- **Vattentjänster**: vatten (V), spillvatten (S), dagvatten från fastighet (Df) och dagvatten från allmän platsmark (Dg).
- **Tomtyta**: fastighetens area som används i tomtyteavgift.
- **Bostadsenhet**: avgiftsgrundande bostadsenhet/lägenhet enligt taxan.
- **Förbindelsepunkt**: punkt där fastigheten ansluts till den allmänna VA-anläggningen.
- **Import/export-paket**: ett JSON-baserat paket som kan flytta all användardata mellan enheter och bevarar full audit trace.
- **Arbetsutrymme**: ett enda gemensamt lokalt arbetsutrymme per installation/webbläsare, utan konton eller profilväxling.
- **Lagring**: endast klientlagring i webbläsaren (IndexedDB och localStorage); ingen server-side persistens.
- **Backend**: ingen backend; all funktionalitet körs i klienten.
